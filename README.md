
# React_Social

React_Social consiste em uma aplicação full stack de uma rede social semelhante ao facebook. 
- Front-end contruído com React
- Back-end construído NodeJS e MongoDB
## Tecnologias utilizadas

- React.js
- CSS3
- Socket.io
- Node.js
- MongoDB
- Express
